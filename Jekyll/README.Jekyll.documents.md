# Document structure
## Folder setup
```txt
assets/
  documents/
   report/
    2026-03-01_General Assembly_Meeting.pdf
    2025-11-12_Board Meeting_Meeting.docx
  invitation/
    2026-02-15_General Assembly_Invitation.pdf
  budget/
    2026-01-01_Budget proposal.pdf

_documents/ ← autogenerated (empty at start)
_includes/
  latest_documents.html
_layouts/
  document.html
plugins/
  document_generator.rb
documents/
  index.md ← list of all documents
  categories.md ← list of categories
```
## Collection
add to _config.yml
```yaml
collections:
  documents:
    output: true
    permalink: /documents/:category/:slug/
```
## plugin file
place here _plugins/document_generator.rb
